# Lab_1_Ivanchenko_appr

Програма реалізує обрахунок матриці та знаходить:
* Обернену матрицю;
* Рішення для системи алгебраїчних рівнянь;
* Ранг матриці;

Програма розбита на такі компоненти як:
* 'InverseMatrixCalculator' - основний клас, який містить в собі методи для обчислення Жорданових виключень.
* 'Matrix' - клас матриці який містить допоміжний метод Clone().
* 'SystemCalculator' - реалізує пошук рішень для системи алгебраїчних рівнянь;
* 'Program' - основна програма;
